## ✨ Features

- **Two product categories** — Hair Extensions & Wigs
- **Shopping cart** — Add, remove, update quantity (saved in localStorage)
- **Sidebar filters** — Category, Hair Type, Length, Price, Origin
- **Customer accounts** — Register & login with welcome discount code
- **WhatsApp support** — Floating chat button for 24/7 support
- **Mobile responsive** — Hamburger menu for all screen sizes
- **Scroll animations** — Smooth fade-in effects
- **Newsletter signup** — With 15% discount offer
- **Bundle deals** — Special pricing on multi-product bundles
